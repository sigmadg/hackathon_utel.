How to animate elements...

You may notice we have used some nice animation effects in this template. here is demo page of animations that you can use in this template. these animations are only supported by modern browsers including IE10+, Firefox, Google chrome, Safari, Opera. animations will be run when animated object displayed in viewport.

To add animation to any element, follow these steps:

  1. add class `animated` to element.
  2. add class set your desired animation

<div class="fadeIn animated">
